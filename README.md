# Guard::Kitchen

Uses test-kitchen to help you develop Chef cookbooks.

## Installation

Add this line to your application's Gemfile:

    gem 'guard-kitchen'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install guard-kitchen

## Usage

From within a cookbook:

    $ guard init kitchen
